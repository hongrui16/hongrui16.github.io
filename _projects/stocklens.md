**StockLens** is a hobby project built out of personal interest in financial markets. 
It is a Flask-based stock analysis tool supporting both A-share (Chinese) and US markets, 
featuring AI-powered analysis using DeepSeek and GPT-4o-mini, 
with a watchlist system for tracking favorite stocks.
